Mammogram Enhancement and Mass Detection in Breast Tissue

Detecting boundaries of masses in mamograms is essential for clinincal purposes. This can be achieved with simple image processing techniques. In this project I have enhanced mammorgram images by applying the Laplacian filter and Histogram Equalization. This was followed by segmenting the image through k-means clustering and then using the Canny Edge detector to locate the mass. (Code)

Tags: Image Processing, Medical Image Analysis

Tech used: Python, OpenCV, SciPy

New Algorithm for Calculating Minimum Vertex Cover

A vertex cover of a graph is a set of vertices that includes at least one endpoint of every edge of the graph. Finding a minimum vertex cover is a classic optimization problem in computer science and is a typical example of an NP-hard optimization problem. I have implemented standard algorithms to find the vertex cover and improved an algorithm suggested by Dahiya et al. (ICMIRA 3013) (Code)

Tags: Graph Theory

Tech used: Java
